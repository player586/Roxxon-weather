# Weather App 🌤️

A simple weather app that shows the current temperature and weather conditions for any city using the OpenWeatherMap API.

---

## Features
- Displays temperature in Fahrenheit (or Celsius if you change units)
- Shows weather condition (Clear, Rain, Clouds, etc.)
- Lets users search for any city
- Simple and clean interface with Times New Roman font and red text
